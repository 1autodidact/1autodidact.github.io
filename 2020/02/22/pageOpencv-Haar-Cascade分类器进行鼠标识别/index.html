检测思路

  1. 计算积分图，使用积分图（Integral Image）对Haar-like进行特征求值
  2. 挑选最优分类器；
  3. 使用AdaBoost算法把这些分类器训练成一个强分类器，用于区分鼠标和非鼠标
  4. 级联，也就是强分类器的强强联手。把强分类器级联到一起，提高准确率

检测方法

利用haar特征，结合级联分类器进行鼠标检测： Haar分类器 = Haar-like特征 + 积分图方法 + AdaBoost +级联

检测目标选择

选择特征明显的物体(颜色和轮廓)

样本构建方法

样本分为正样本(有目标)、负样本(无目标)
